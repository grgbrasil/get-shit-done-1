**Internal helpers:**
- `registryDir(cwd)` -- returns `path.join(planningRoot(cwd), 'ops')`
- `registryPath(cwd)` -- returns `path.join(registryDir(cwd), REGISTRY_FILENAME)`
- `readRegistry(cwd)` -- reads registry.json, returns `{ areas: [] }` if not exists
- `writeRegistry(cwd, registry)` -- writes registry.json with `JSON.stringify(registry, null, 2)`, creates dir with `{ recursive: true }` if missing
- `areaDir(cwd, slug)` -- returns `path.join(registryDir(cwd), slug)`
- `ensureAreaDir(cwd, slug)` -- creates area dir if missing, returns path
- `slugify(text)` -- delegates to `generateSlugInternal(text)`, per D-05

**File listing helper:**
- `listProjectFiles(cwd)` -- uses `execSync('git ls-files', { cwd })` to get file list (respects .gitignore). Falls back to recursive `fs.readdirSync` if git fails. Excludes `node_modules/`, `dist/`, `build/`, `.planning/`, `.git/`.

**Detection functions:**
- `detectFramework(cwd, files)` -- iterates FRAMEWORK_PATTERNS, checks if any pattern's files exist in the file list. Returns array of `{ framework, type, files: matchedFiles }`.
- `detectAreasFromRoutes(cwd, frameworkMatch)` -- reads matched route files, applies routeRegex, extracts top-level path segments as area names. Returns array of `{ name, slug, detected_by: 'route', confidence: 'high', files: [routeFile] }`.
- `detectAreasFromDirectories(cwd, frameworkMatch)` -- scans directories in pattern, each immediate subdirectory becomes an area. Returns array of `{ name, slug, detected_by: 'directory', confidence: 'medium', files: [] }`.
- `deduplicateAreas(routeAreas, dirAreas)` -- merge by slug. If same slug from both sources, keep one entry with `source: "auto"`, `detected_by: ["route", "directory"]`. Route name takes priority.

**Command functions (per D-17):**

`cmdOpsInit(cwd, args, raw)`:
1. Call `listProjectFiles(cwd)` to get file list
2. Call `detectFramework(cwd, files)` for each FRAMEWORK_PATTERNS entry
3. For route-type matches, call `detectAreasFromRoutes()`
4. For directory-type matches, call `detectAreasFromDirectories()`
5. Call `deduplicateAreas()` to merge results
6. Build registry object: `{ areas: [...], created_at: new Date().toISOString(), last_scanned: new Date().toISOString() }`
7. Each area entry per D-02: `{ slug, name, source: "auto", detected_by, confidence, created_at, last_scanned, components_count: 0 }`
8. Call `writeRegistry(cwd, registry)`
9. Call `output({ success: true, areas_detected: registry.areas.length, areas: registry.areas }, raw)`

`cmdOpsAdd(cwd, areaName, args, raw)`:
1. If !areaName, call `error('Usage: gsd-tools ops add <area-name>')`
2. Slugify the name via `slugify(areaName)`
3. Read existing registry, check if slug already exists -- if yes, `error('Area already exists: ' + slug)`
4. Create entry: `{ slug, name: areaName, source: "manual", detected_by: "manual", confidence: "high", created_at: new Date().toISOString(), last_scanned: null, components_count: 0 }`
5. Push to registry.areas, writeRegistry
6. Call `ensureAreaDir(cwd, slug)` -- create per-area directory immediately for manual areas
7. `output({ success: true, area: entry }, raw)`

`cmdOpsList(cwd, raw)`:
1. Read registry
2. `output({ areas: registry.areas || [] }, raw)`

`cmdOpsGet(cwd, area, raw)`:
1. If !area, `error('Usage: gsd-tools ops get <area>')`
2. Read registry, find by slug (slugify input first)
3. If not found, `error('Area not found: ' + slug)`
4. Return the entry plus check if tree.json exists in area dir: `output({ ...entry, has_tree: fs.existsSync(path.join(areaDir(cwd, slug), 'tree.json')) }, raw)`

`cmdOpsMap(cwd, area, raw)`:
- Stub for now (Plan 02 implements): `error('ops map not yet implemented -- see Plan 05-02')`
