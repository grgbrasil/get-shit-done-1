       - Extract goal from section via regex `\*\*Goal\*\*:?\s*([^\n]+)`

       - If `config.workflow?.ui_phase === false` OR `config.workflow?.ui_safety_gate === false`, return immediately
       - If `!roadmapPhase?.section`, return immediately
       - If `!hasUiIndicators(roadmapPhase.section)`, return immediately
       - Check for UI-SPEC.md in phase directory: `fs.existsSync(path.join(planningRoot(cwd), phaseInfo?.directory || '', phase + '-UI-SPEC.md'))` — also check for files matching `*UI-SPEC*` pattern in the phase directory
       - If UI-SPEC.md not found, push blocker: `{ type: 'missing_ui_spec', message: 'Phase ${phase} appears to have UI work but no UI-SPEC.md found.', action: 'run_command', command: '/gsd:ui-phase ${phase}', severity: 'warning', skippable: true }`

    6. Implement `checkDependentPhasesComplete(cwd, phase, roadmapPhase, blockers)`:
       - Parse `roadmapPhase?.section` for `**Depends on:**` line
       - If matches `nothing|none|n\/a`, return immediately
       - Extract phase numbers via regex `\d+(?:\.\d+)*/g`
       - Track visited phases in a Set to prevent circular dependency loops
       - For each dependency, call `findPhaseInternal(cwd, dep)` to check disk_status
       - A phase is complete when it has plans AND all plans have matching summaries (plans.length > 0 AND plans.length === summaries.length)
       - If incomplete, push blocker: `{ type: 'incomplete_dependency', message: 'Phase ${dep} must complete before Phase ${phase}', action: 'run_command', command: '/gsd:execute-phase ${dep}', severity: 'blocking', skippable: false }`

    7. Implement `checkPlansExist(cwd, phase, phaseInfo, blockers)`:
       - If `(phaseInfo?.plans?.length || 0) > 0`, return immediately
       - Push blocker: `{ type: 'no_plans', message: 'No plans found for Phase ${phase}. Run plan-phase first.', action: 'run_command', command: '/gsd:plan-phase ${phase}', severity: 'blocking', skippable: false }`

    8. Implement `cmdPreflight(cwd, command, phase, raw)`:
       - Validate command is one of: 'plan-phase', 'execute-phase', 'ui-phase'
       - Call `loadConfig(cwd)`, `findPhaseInternal(cwd, phase)`, `getRoadmapPhaseInternal(cwd, phase)`
       - Build `blockers = []`
       - For `plan-phase`: call checkContextExists, checkUiSpec, checkDependentPhasesComplete (in this order)
       - For `execute-phase`: call checkPlansExist, checkDependentPhasesComplete
       - For `ui-phase`: call checkContextExists
       - Get firstBlocker = blockers[0] || null
       - Call `output({ ready: blockers.length === 0, blockers, next_action: firstBlocker?.action || null, next_command: firstBlocker?.command || null, phase_number: phase, command_checked: command }, raw)`

    9. Export: `module.exports = { cmdPreflight }`
